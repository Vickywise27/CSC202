01 What is Java?

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle) in 1995, created by James Gosling.

Key principle: "Write Once, Run Anywhere" (WORA) — Java code compiles to bytecode which runs on any machine with a Java Virtual Machine (JVM).

Key features of Java:

  • Platform-independent — runs on any OS via the JVM
  • Object-Oriented — everything is organized into classes and objects
  • Strongly typed — every variable must have a declared type
  • Secure — built-in security features
  • Robust — strong memory management, exception handling
  • Multithreaded — can run multiple tasks simultaneously
JDK vs JRE vs JVM:
• JVM (Java Virtual Machine) — executes bytecode
• JRE (Java Runtime Environment) — JVM + libraries to run Java programs
• JDK (Java Development Kit) — JRE + tools to write and compile Java programs

02 Basic Program Structure

Every Java program must have at least one class and a main method — the entry point of the program.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Breaking it down:

  • public class HelloWorld — defines a class named HelloWorld. The filename must match: HelloWorld.java
  • public static void main(String[] args) — the main method. Program starts here.
  • System.out.println() — prints output to the console with a new line
  • System.out.print() — prints without a new line
Important rules:
• Java is case-sensitive (Main ≠ main)
• Every statement ends with a semicolon ( ; )
• Code blocks are enclosed in curly braces { }
• The class name must match the filename exactly

Execution flow: Source code (.java) → Compiler (javac) → Bytecode (.class) → JVM → Output

04 Declaring & Initializing Variables

A variable is a named storage location in memory that holds a value.

Syntax: dataType variableName;    or    dataType variableName = value; 
// Declaration only
int studentCount;

// Declaration + Initialization
int studentCount = 50;
double price = 99.99;
boolean isActive = true;
char grade = 'A';
String name = "James";

Types of variables in Java:

  • Local variable — declared inside a method; only accessible within that method
  • Instance variable — declared inside a class but outside methods; each object has its own copy
  • Static/Class variable — declared with the static keyword; shared by all objects of the class

Naming rules for variables:

  • Must start with a letter, underscore (_), or dollar sign ($)
  • Cannot start with a number
  • Cannot be a Java keyword (int, class, void, etc.)
  • Case-sensitive (age ≠ Age)
  • Convention: use camelCase (e.g. studentName, totalPrice)
Constants: Use the final keyword to make a variable constant (its value cannot change).
Convention: use ALL_CAPS for constants.
final double PI = 3.14159;

System.out.printf("Age: %d", age);  // formatted output

Input — Scanner class:

import java.util.Scanner;  // must import at top

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter your name: ");

07 Type Casting

Type casting is converting a value from one data type to another.

Widening (Implicit) Casting — done automatically, no data loss.
Order: byte → short → int → long → float → double 
int x = 10;
double y = x;  // automatic — int to double
System.out.println(y);  // 10.0
Narrowing (Explicit) Casting — must be done manually; may lose data. 
double a = 9.99;
int b = (int) a;  // manual cast — truncates decimal
System.out.println(b);  // 9 (not rounded!)

Narrowing goes in the reverse direction: double → float → long → int → short → byte

String conversions:
• int to String: String s = String.valueOf(42); or "" + 42
• String to int: int n = Integer.parseInt("42");
• String to double: double d = Double.parseDouble("3.14");

08 Comments & Good Practice

Comments are notes in code that the compiler ignores. They explain what the code does.

// This is a single-line comment

/* This is a
   multi-line comment */

/**
 * This is a Javadoc comment
 * Used to document classes and methods
 */
public class Example {
    public static void main(String[] args) {
        int x = 5; // stores the number 5
    }
}

Quick recap — things to always remember:

  • Java is case-sensitive
  • Every statement ends with ;
  • Class name must match filename
  • Program starts at main()
  • Use import for external classes (like Scanner)
  • Primitive types are lowercase (int, double); String starts with capital S
  • = assigns, == compares
  • Integer division truncates: 7/2 = 3, not 3.5
